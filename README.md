Setup
=============

MacOS

For Unix-based (make), install premake4 and use
premake4 gmake

Install XQuartz and log out, log in again.
https://www.xquartz.org/
-------------

Run
=============
Run file ./run.sh

Game Usage:

Press R/r to toggle between auto-run mode and manual mode.
In manual mode, press SPACE to step.
