Setup on MacOS
=============

- In root directory, run:
premake4 gmake

- Install XQuartz (https://www.xquartz.org/) after that, log out and log in again.


Run
=============
Run file ./run.sh

Game Usage:

Press R/r to toggle between auto-run mode and manual mode.
In manual mode, press SPACE to step.
